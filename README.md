## Prerequisites

- Node.JS
- Node Package Manager

## Installation

How to clone and startup the server:
```bash
git clone https://github.com/MedHubTeam/MedHub-Backend.git
cd MedHub-Backend
npm install
```

## Usage

Start the server:
```bash
npm run start
```

Run unit tests:
```bash
npm run test
```

Run eslint:
```bash
npm run lint
```
